# Security Policy

## Supported Versions

The latest release line is supported for security fixes.

## Sensitive Data Warning

This project handles attendance, employee, and payroll information. Do not commit:

- Real employee names, IDs, phone numbers, or addresses
- Real payroll records
- Production JSON backups
- Spreadsheet exports containing personal or salary data
- Screenshots with private employee information

Use anonymized test data only.

## Reporting a Vulnerability

Please open a private security advisory on GitHub if available, or contact the maintainer directly.

Include:

- A clear description
- Impact
- Reproduction steps
- Suggested fix, if known

Do not publicly disclose vulnerabilities involving payroll data exposure until a fix is available.
