# 出勤薪資管理系統 規格書

> 版本：v2.1  
> 更新日期：2026-05-06  
> 形式：單一 HTML 離線執行檔

---

## 1. 系統定位

本系統提供行政、人資、會計或主管在單機環境中管理出勤與薪資。主程式維持單一檔案：

```text
出勤薪資管理系統.html
```

系統資料預設儲存在瀏覽器本機空間，使用者需要定期匯出 `JSON` 備份，避免換電腦、清除瀏覽器資料或系統異常時遺失資料。

主要功能包含：

- 員工資料管理
- 出勤、遲到、早退與加班計算
- 請假 / 補休管理
- 自訂國定假日
- 勞保、健保、勞退參考表套用與手動覆寫
- 月底薪資結算
- 當月全體薪資總額統計
- 薪資單列印
- `.xlsx` 匯出
- 出勤異常與漏 KEY 警示
- `JSON` 備份與還原

## 3. 員工資料

每位員工資料包含：

- 姓名
- 薪資類型：月薪制 / 時薪制
- 月薪或時薪金額
- 到職日
- 離職日
- 固定加給
- 勞保自付額
- 健保自付額
- 勞退提繳
- 其他扣款
- 勞健保扣款來源：使用參考表 / 手動輸入
- 身心障礙等級
- 特休手動調整
- 備註
- 在職 / 離職狀態

### 3.2 未足月薪資規則

月薪員工若在結算月份內發生以下情況，月薪基數需按比例計算：

- 月中到職
- 月中離職
- 同月到職且同月離職

此處月薪基數定義為：

```text
底薪 + 固定加給
```

比例公式統一採：

```text
(底薪 + 固定加給) ÷ 30 × 在職天數
```

規則補充：

- 離職日包含在計薪天數內。
- 未足月月份的遲到 / 早退扣薪、請假扣薪、週六加班費、週日加發 1 日工資、國定假日加發 1 日工資，皆使用同一個比例後月薪基數換算。
- 薪資結算、報表、薪資單與 `.xlsx` 匯出都需顯示未足月說明。
- 離職員工在離職後未來月份不列入結算，但歷史月份仍可查詢、列印與匯出。

### 3.1 固定加給

固定加給屬於每月固定發放項目。月薪員工的固定加給會納入：

- 薪資結算應發金額
- 週六加班費計算基礎
- 國定假日加發 1 日工資計算基礎

月薪員工加班費基礎採：

```text
(月薪 + 固定加給) / 30 / 8 / 60
```

## 4. 標準工時

### 4.1 平日標準班

- 上班時間：`08:00`
- 下班時間：`17:00`
- 午休時間：`12:00 ~ 13:00`
- 正常工時：`8 小時`

平日 `08:00 ~ 17:00` 會扣除中午 1 小時休息，因此不可顯示為 `7.67 小時`。若出勤區間只和午休部分重疊，僅扣除實際重疊分鐘數。

### 4.2 計薪區間

| 區間 | 是否計薪 | 說明 |
|---|---|---|
| `08:00` 前 | 否 | 提早到班不另計薪 |
| `08:00 ~ 17:00` | 是 | 扣除 `12:00 ~ 13:00` 午休後為 `8 小時` |
| `17:00 ~ 18:00` | 否 | 不算正常工時，也不算加班 |
| `18:00` 後 | 是 | 依加班倍率計算 |

## 5. 遲到與早退

### 5.1 遲到

- `08:01` 起視為遲到
- 遲到分鐘數只計 `08:00 ~ 17:00` 的標準工時，午休 `12:00 ~ 13:00` 不納入
- 若同日有假勤紀錄，請假分鐘會先抵扣遲到分鐘

### 5.2 早退

- `17:00` 前離開視為早退
- 早退分鐘數只計 `08:00 ~ 17:00` 的標準工時，午休 `12:00 ~ 13:00` 不納入
- 若同日有假勤紀錄，剩餘請假分鐘會抵扣早退分鐘

午休邊界範例：

```text
07:51 ~ 12:02
正常工時 = 08:00 ~ 12:02 扣除 12:00 ~ 12:02 = 240 分鐘 = 4 小時
早退分鐘 = 12:02 ~ 17:00 扣除 12:02 ~ 13:00 = 240 分鐘
```

### 5.3 半日請假搭配出勤

若員工上午請假、下午正常上班，系統會以請假紀錄抵扣上午缺勤，不會再把下午 `13:00` 上班判定為整個上午遲到。

範例：

```text
08:00 ~ 12:00 喪假 0.5 天
12:00 ~ 13:00 午休
13:00 ~ 17:00 正常上班
結果：正常給薪工時 8 小時，遲到扣薪 0 元
```

### 5.4 扣薪公式

月薪制每分鐘扣薪：

```text
月薪 / 30 / 8 / 60
```

時薪制每分鐘扣薪：

```text
時薪 / 60
```

## 6. 加班規則

### 6.1 平日加班

- 平日加班從 `18:00` 後開始計算
- 前 2 小時：`1.34 倍`
- 超過 2 小時：`1.67 倍`

### 6.2 週六加班（休息日加班）

週六加班採有效計薪區間，不是把打卡全時段全部算入。

有效計薪規則：

- `08:00` 前提早到班不計薪
- `08:00 ~ 17:00` 視為週六加班時段
- `12:00 ~ 13:00` 午休不計
- `17:00 ~ 18:00` 不計薪
- `18:00` 後再依加班時數累加

週六加班倍率：

- 前 2 小時：`1.34 倍`
- 超過 2 小時：`1.67 倍`

週六案例：

```text
月薪 31000 + 固定加給 2000 = 計算基礎 33000
33000 / 30 / 8 / 60 * 1.34 * 120 = 368
33000 / 30 / 8 / 60 * 1.67 * 360 = 1378
週六 8 小時加班費 = 1746
```

另一個週六案例：

```text
月薪 30000，07:48 上班，17:16 下班
08:00 前不計薪，17:00 ~ 18:00 不計薪
有效計薪工時仍為 8 小時
30000 / 30 / 8 / 60 * 1.34 * 120 = 335
30000 / 30 / 8 / 60 * 1.67 * 360 = 1252.5
四捨五入後加班費 = 1588
```

### 6.3 週日加班（例假日）

週日加班目前固定為：

- 加發 `1 日工資`
- 補休 `8 小時`

### 6.4 國定假日加班

國定假日加班可選處理方式：

- 加發 `1 日工資`
- 補休 `8 小時`

選擇 `加發 1 日工資` 時：

- 月薪制：加發 `(月薪 + 固定加給) / 30`
- 時薪制：加發 `時薪 * 8`
- 不另外累積補休

選擇 `補休 8 小時` 時：

- 不加發 1 日工資
- 補休餘額增加 `8 小時`

## 7. 請假與補休

### 7.1 假別單位

| 假別 | 輸入單位 | 薪資影響 |
|---|---|---|
| 特別休假 | 天 | 不扣薪 |
| 事假 | 分鐘 | 全額扣薪 |
| 病假 | 分鐘 | 半薪扣款 |
| 生理假 | 分鐘 | 比照病假半薪 |
| 喪假 | 天 | 不扣薪 |
| 補休使用 | 分鐘 | 不扣薪，扣補休餘額 |
| 其他假別 | 天 | 不扣薪 |

### 7.2 事假 / 病假 / 生理假

`事假`、`病假`、`生理假` 以分鐘輸入。系統會同步更新：

- 出勤紀錄頁的請假分鐘數
- 出勤紀錄頁的應扣金額
- 薪資結算的請假扣薪
- 報表與 `.xlsx` 匯出
- 同日出勤的遲到 / 早退抵扣

扣薪公式：

```text
事假扣薪 = 每分鐘薪資 * 請假分鐘
病假扣薪 = 每分鐘薪資 * 請假分鐘 * 0.5
生理假扣薪 = 每分鐘薪資 * 請假分鐘 * 0.5
```

時薪制員工以實際正常上班分鐘與有薪假可給薪分鐘計薪，事假不再額外扣一次薪資，以避免「未給薪又扣薪」的重複扣款。

## 8. 國定假日與自訂例外

系統會內建抓取常見國定假日，但仍支援人工自訂，以處理公司補班、臨時公告或例外假日。

自訂國定假日可用於：

- 新增公司特定休假日
- 調整補班日
- 修正年度政策例外
- 讓出勤類型與薪資結算能依公司實際規則處理

## 9. 勞健保與勞退

### 9.1 參考表模式

系統支援載入：

- `2025` 參考表
- `2026` 參考表

參考表每列包含：

- 生效日期
- 薪資下限
- 薪資上限
- 勞保自付
- 健保自付
- 勞退提繳
- 備註

### 9.2 自動套用邏輯

若員工保險來源選擇 `使用參考表`：

- 先依結算月份找出生效中的參考表
- 再依員工薪資落點尋找級距
- 自動帶入勞保、健保與勞退提繳

### 9.3 手動模式與臨時修改

若員工保險來源選擇 `手動輸入`：

- 勞保、健保、勞退以員工資料內填寫值為主
- 不再自動套用參考表

若員工原本使用參考表，但需要臨時修改，可按：

```text
改用手動編輯目前數值
```

系統會先帶入目前參考表數值，再解鎖欄位供人工修改。

### 9.4 身心障礙補助

系統支援依身心障礙等級調整勞保 / 健保實扣金額：

| 等級 | 勞保 / 健保實扣比例 |
|---|---|
| 無 | 100% |
| 輕度 | 75% |
| 中度 | 50% |
| 重度 | 0% |
| 極重度 | 0% |

說明：

- 僅調整勞保 / 健保
- 勞退提繳與其他扣款不受此比例影響

## 10. 月結手動項目

薪資結算支援每月針對個別員工調整下列項目：

| 欄位 | 加減方向 | 說明 |
|---|---|---|
| 伙食津貼 | 加項 | 公司補貼給員工 |
| 伙食代墊 | 減項 | 公司先代墊，月底向員工扣回 |
| 其他費用 | 減項 | 其他需扣回或代扣費用 |
| 薪資加項 | 加項 | 彈性增加，例如獎金、補貼、調整款 |
| 薪資減項 | 減項 | 彈性扣除，例如借支、罰款、調整款 |
| 年終獎金 | 加項 | 額外獎金欄位 |
| 勞保覆寫 | 減項 | 覆寫當月勞保扣款 |
| 健保覆寫 | 減項 | 覆寫當月健保扣款 |
| 勞退覆寫 | 公司提繳資訊 | 覆寫當月勞退提繳 |
| 其他扣款覆寫 | 減項 | 覆寫員工資料中的其他扣款 |

### 10.1 品項備註

以下欄位可分別填寫品項備註：

- 伙食津貼備註
- 伙食代墊備註
- 其他費用備註
- 薪資加項備註
- 薪資減項備註
- 月結總備註

備註會合併顯示於：

- 薪資結算
- 正式薪資單
- 報表中心
- `.xlsx` 匯出

### 10.2 薪資結算總額

薪資結算頁按下 `計算薪資` 後，報表上方需顯示本月全體統計：

- 結算員工數
- 全體實發薪資總額
- 應發合計
- 扣款合計

## 11. 出勤異常與漏 KEY 警示

報表中心會提供異常清單，用來協助發薪前檢查資料。

目前包含：

- 遲到
- 早退
- 加班
- 請假扣薪
- 正常上班日漏 KEY 警示

### 11.1 正常上班日漏 KEY 警示

若符合以下條件，系統會提示可能漏登：

- 員工為在職
- 日期為一般正常應上班日
- 當日沒有出勤紀錄
- 當日沒有請假 / 補休紀錄

此警示用來提醒人工疏漏，不會直接產生扣薪；使用者需回到出勤或假勤頁確認後再處理。

## 12. 報表與輸出

### 12.1 報表中心

支援報表：

- 員工月薪資明細
- 公司月總表
- 出勤異常清單
- 員工月統計

### 12.2 正式薪資單

正式薪資單會顯示：

- 出勤摘要
- 應發項目
- 扣款項目
- 伙食津貼 / 伙食代墊 / 其他費用
- 薪資加項 / 薪資減項
- 品項備註
- 保險來源
- 勞退提繳
- 簽名欄

### 12.3 `.xlsx` 匯出

`.xlsx` 匯出包含：

- 薪資總表
- 薪資單
- 出勤明細
- 異常清單

匯出資料會帶出：

- 保險來源
- 勞退提繳
- 請假扣薪
- 伙食津貼 / 伙食代墊 / 其他費用
- 薪資加項 / 薪資減項
- 品項備註

## 13. 備份與還原

使用者需定期執行：

- 匯出 `JSON` 備份
- 將備份放到公司資料夾、雲端或外接硬碟
- 換電腦或瀏覽器前，先確認備份可匯入

建議發薪後立即備份一次，以保留當月結算依據。
