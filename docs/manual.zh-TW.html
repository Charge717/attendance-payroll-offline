出勤薪資管理系統 新手操作手冊

這份手冊是寫給不熟悉電腦操作的人看的。請把它當成每日工作流程小抄：先照案例做，熟了以後再回頭看規則。

手冊版本 v1.7
系統形式 單一 HTML 離線檔
更新日期 2026-04-29
先記住 6 件事：
  1. 資料存在目前這台電腦的瀏覽器內，請每月匯出 JSON 備份。
  2. 第一次建置前，請先載入 2025 或 2026 勞健保參考表。
  3. 平日 08:00 ~ 17:00，扣掉午休後正常工時是 8 小時。
  4. 事假、病假、生理假 可以用 分鐘 輸入，會反映到出勤與薪資。
  5. 週六加班會把固定加給納入基礎，且 08:00 前 與 17:00 ~ 18:00 不計薪。
  6. v1.7 修正午休部分重疊時誤扣整段午休的問題，例如 12:02 下班。
操作建議：每次月底發薪前，先看「出勤異常清單」，再進入薪資結算。這樣比較不容易漏掉未登出勤、請假扣薪或手動調整。

二、第一次建置前先載入參考表

情境
你第一次打開系統，要先讓勞保、健保、勞退有參考級距，之後新增員工才可以自動帶入正確數值。
  1. 打開 出勤薪資管理系統.html。
  2. 切到 員工管理 或勞健保參考表所在區塊。
  3. 如果要使用 2025 年級距，按 載入 2025 參考表。
  4. 如果要使用 2026 年級距，按 載入 2026 參考表。
  5. 確認參考表出現薪資下限、薪資上限、勞保、健保、勞退等欄位。
  6. 之後新增員工時，保險來源建議先選 使用參考表。
注意：如果政策改變，可以修改參考表；使用參考表的員工在月結時會依薪資級距自動帶入。若只有某位員工臨時不同，請不要改整張表，改用該員工的手動模式。

三、案例 1：建立員工資料

情境
新進一位月薪員工，月薪 31000，固定加給 2000。你要建立資料，讓日後薪資、保險與加班費都能自動計算。
  1. 切到 員工管理。
  2. 按 新增員工。
  3. 輸入姓名、薪資類型、月薪或時薪、到職日。
  4. 月薪欄輸入 31000。
  5. 固定加給欄輸入 2000。
  6. 勞健保扣款來源先選 使用參考表。
  7. 確認身心障礙等級，沒有就選 無。
  8. 按 儲存。
固定加給會影響什麼？

固定加給會納入月薪員工的薪資結算，也會納入週六加班費與國定假日加發 1 日工資的計算基礎。

31000 + 2000 = 33000 週六加班每分鐘基礎 = 33000 / 30 / 8 / 60

臨時修改勞保 / 健保 / 勞退

  1. 打開員工編輯視窗。
  2. 如果目前保險來源是 使用參考表，欄位會灰階，這是正常保護機制。
  3. 按 改用手動編輯目前數值。
  4. 系統會先把目前參考表算出的數值帶入欄位。
  5. 欄位解鎖後，再手動修改勞保、健保或勞退。
  6. 按 儲存。

四、案例 2：每天記錄上下班

情境
員工正常上班，早上 08:00 打卡，下午 17:00 下班。你要確認系統顯示為 8 小時。
  1. 切到 出勤紀錄。
  2. 選擇正確月份。
  3. 按 新增出勤紀錄。
  4. 選員工與日期。
  5. 上班打卡輸入 08:00。
  6. 下班打卡輸入 17:00。
  7. 出勤類型選 平日。
  8. 檢查下方 預覽計算結果，正常工時應為 8 小時。
  9. 確認無誤後按 儲存。
為什麼不是 9 小時？

因為 12:00 ~ 13:00 是午休，不計入工時，所以 08:00 ~ 17:00 正常工時為 8 小時。

五、案例 3：週六加班怎麼算

情境 A
月薪 31000，固定加給 2000，週六 08:00 到 17:00 上班。你要確認加班費是否以 33000 當基礎。
  1. 切到 出勤紀錄。
  2. 按 新增出勤紀錄。
  3. 選員工與週六日期。
  4. 上班打卡輸入 08:00，下班打卡輸入 17:00。
  5. 出勤類型選 休息日加班（週六）。
  6. 確認預覽結果顯示週六加班 8 小時。
計算基礎 = 31000 + 2000 = 33000 前 2 小時：33000 / 30 / 8 / 60 * 1.34 * 120 = 368 後 6 小時：33000 / 30 / 8 / 60 * 1.67 * 360 = 1378 加班費合計 = 1746
情境 B
月薪 30000，週六 07:48 上班、17:16 下班。提早到與 17:00 後 16 分鐘都不應多算。
08:00 前不計薪 12:00 ~ 13:00 午休不計薪 17:00 ~ 18:00 不計薪 有效計薪工時 = 8 小時 加班費四捨五入 = 1588

六、案例 4：事假分鐘與半日喪假

情境
員工上午請假、下午正常上班。你要讓請假分鐘抵扣上午缺勤，下午正常上班的分鐘仍正常給薪。
  1. 切到 假勤管理。
  2. 按 新增請假 / 補休使用。
  3. 選員工與日期。
  4. 若是假別選 事假，使用分鐘數輸入 240 分鐘。
  5. 若是假別選 喪假，天數輸入 0.5 天，並填寫喪假級距與案件代號。
  6. 備註可寫，例如 上午請假。
  7. 按 儲存。
  8. 回到 出勤紀錄，新增同一天 13:00 ~ 17:00 的正常出勤。
  9. 預覽應顯示下午正常工時，不應把上午請假時段列為遲到扣薪。
  10. 月底到 薪資結算，確認事假扣薪或喪假不扣薪已被帶入。
假別 輸入單位 扣薪方式
事假 分鐘 全額扣薪
病假 分鐘 半薪扣款
生理假 分鐘 比照病假半薪
補休使用 分鐘 不扣薪，扣補休餘額

七、案例 5：國定假日加班選一日工資或補休

情境
員工在國定假日出勤。公司要依當次狀況決定加發 1 日工資，或改用補休 8 小時。
  1. 切到 出勤紀錄。
  2. 按 新增出勤紀錄。
  3. 選員工、日期、上班時間、下班時間。
  4. 出勤類型選 國定假日加班。
  5. 在國定假日處理方式選擇 加發 1 日工資 或 補休 8 小時。
  6. 查看預覽結果。
  7. 確認後按 儲存。
怎麼選？

如果公司要給錢，選 加發 1 日工資。如果公司與員工約定改給補休，選 補休 8 小時。兩者擇一，避免同一天同時多給薪又多給補休。

八、案例 6：月底輸入伙食、其他費用與薪資加減項

情境
月底結算時，某員工有伙食津貼 1500、伙食代墊 300、其他費用 200，另有一次性獎金 1000。你要把金額與品項寫清楚，讓薪資單看得懂。
  1. 切到 薪資結算。
  2. 選月份後按 計算薪資。
  3. 先確認上方 全體實發薪資總額、應發合計與扣款合計是否合理。
  4. 找到該員工，進入月結調整欄位。
  5. 伙食津貼 輸入 1500，備註填 本月餐費補貼。
  6. 伙食代墊 輸入 300，備註填 公司代墊午餐。
  7. 其他費用 輸入 200，備註填 制服清潔費。
  8. 薪資加項 輸入 1000，備註填 專案獎金。
  9. 若有需扣回款項，可使用 薪資減項 並填寫備註。
  10. 儲存後重新確認薪資明細與薪資單。
欄位 加減方向 範例備註
伙食津貼 加項 本月餐費補貼
伙食代墊 減項 公司代墊午餐
其他費用 減項 制服清潔費、宿舍代扣、代買用品
薪資加項 加項 獎金、補貼、差額補發
薪資減項 減項 借支扣回、調整扣款
小提醒：品項備註會同步到薪資單、報表與 Excel 匯出，建議寫得像收據摘要一樣清楚，日後查帳會省很多時間。

九、案例 7：正常上班日漏 KEY 警示

情境
某位在職員工某天是正常上班日，但操作人員忘記輸入出勤，也沒有請假紀錄。系統要在發薪前提醒。
  1. 月底前切到 報表中心。
  2. 選擇要檢查的月份。
  3. 打開 出勤異常清單。
  4. 查看是否有「正常上班日未建立出勤或請假紀錄」類似警示。
  5. 若是真的漏登，回到 出勤紀錄 新增打卡資料。
  6. 若當天其實有請假，回到 假勤管理 補登請假。
  7. 修正後再次回到異常清單確認警示消失。
重要：漏 KEY 警示只是提醒，不會自動扣薪。一定要由使用者確認事實後，再決定補出勤或補請假。

十、月底發薪前建議流程

  1. 先到 報表中心 看 出勤異常清單。
  2. 處理遲到、早退、請假、加班與漏 KEY 警示。
  3. 回到 薪資結算，選月份並按 計算薪資。
  4. 先看上方全體薪資總額，再逐一確認底薪、固定加給、加班費、請假扣薪。
  5. 確認勞保、健保、勞退來源是參考表或手動輸入。
  6. 如需當月調整，輸入伙食津貼、伙食代墊、其他費用、薪資加項或薪資減項。
  7. 每個手動金額盡量填寫品項備註。
  8. 確認薪資單格式與金額無誤。
  9. 匯出 `.xlsx` 或列印正式薪資單。
  10. 最後到 備份管理 匯出 JSON 備份。
月底檢查清單：
  • 有沒有漏登正常上班日？
  • 事假 / 病假是否用分鐘輸入？
  • 週六加班是否以月薪 + 固定加給計算？
  • 國定假日是給一日工資還是補休？
  • 勞健保是參考表還是手動值？
  • 伙食與其他費用是否有備註？

十一、報表、Excel 與薪資單

正式薪資單

正式薪資單適合列印給員工或留存。它會顯示應發、應扣、保險來源、勞退提繳、請假扣薪、伙食與其他費用、品項備註與簽名欄。

Excel 匯出

`.xlsx` 匯出適合交給會計或主管覆核。匯出內容包含薪資總表、薪資單、出勤明細與異常清單。

JSON 備份

JSON 是系統完整資料備份。Excel 是報表，不等於完整備份。發薪後請務必再匯出一次 JSON。

不要只留 Excel：如果以後要把資料匯回系統，請使用 JSON 備份；Excel 主要是查帳與列印用途。

十二、常見問題

1. 為什麼勞保 / 健保沒有自動帶入？

  • 請先確認已按 載入 2025 參考表 或 載入 2026 參考表。
  • 請確認員工保險來源是 使用參考表。
  • 請確認員工薪資落在參考表級距內。

2. 參考表模式欄位灰色不能改，怎麼辦？

  • 按 改用手動編輯目前數值。
  • 系統會先帶入目前參考值，再讓欄位可以手動修改。

3. 為什麼平日 08:00 ~ 17:00 是 8 小時？

  • 因為中午 12:00 ~ 13:00 是休息時間。
  • 9 小時打卡區間扣掉 1 小時午休，等於 8 小時。

4. 為什麼週六 07:48 ~ 17:16 只算 8 小時？

  • 08:00 前 提早到班不計薪。
  • 17:00 ~ 18:00 不計薪。
  • 所以有效計薪時間仍為 08:00 ~ 17:00 扣午休後 8 小時。

5. 為什麼 12:02 下班是早退 240 分鐘？

  • 12:00 ~ 13:00 是午休，不算正常工時，也不算早退。
  • 07:51 ~ 12:02 的正常工時是 08:00 ~ 12:00，共 4 小時。
  • 早退從下午應上班的 13:00 ~ 17:00 計算，共 240 分鐘。

6. 其他費用和薪資減項差在哪裡？

  • 其他費用 適合固定分類的代扣或代墊費用，例如制服、宿舍、代買用品。
  • 薪資減項 保留給彈性調整，例如借支扣回或特殊扣款。
