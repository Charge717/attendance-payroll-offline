# 中文推廣短文

## 一句話

我開源了一個免費、免安裝、離線優先的出勤薪資管理工具，適合台灣小公司、店家、工作室、人資與會計先用瀏覽器管理出勤、假勤、月結薪資與薪資單。

## 短版

我把最近整理的「出勤薪資管理系統」開源了：

https://github.com/Charge717/attendance-payroll-offline

它是一個單一 HTML 的離線工具，不需要伺服器、不需要帳號，資料預設存在自己的瀏覽器本機。功能包含員工資料、出勤紀錄、假勤管理、月薪 / 時薪薪資結算、勞健保與勞退參考表、薪資單列印、Excel 匯出、JSON 備份還原，以及 Demo Data。

目前計算規則以台灣使用情境為主，核心 UI 已支援繁中 / 英文。專案也補上測試、GitHub Actions、隱私掃描與 i18n 稽核。歡迎試用、給建議，覺得有幫助也歡迎給 GitHub Star。

Demo:
https://charge717.github.io/attendance-payroll-offline/

## 長版

很多小公司、店家或工作室其實不一定需要大型雲端人資系統，只是需要一個能記錄出勤、假勤、薪資結算、薪資單與備份的工具。

所以我把「出勤薪資管理系統」整理成一個公開開源版本：Attendance Payroll Offline。

它的特色是：

- 免費、MIT 開源
- 單一 HTML，瀏覽器直接開啟
- 不需要伺服器或資料庫
- 員工與薪資資料預設存在本機瀏覽器
- 支援 JSON 備份 / 還原
- 支援出勤、假勤、月結薪資、薪資單列印、Excel 匯出
- 內建 fictional Demo Data，方便安全試用
- 補上薪資計算回歸測試、隱私稽核與 i18n 稽核

目前核心 UI 支援繁中與英文，但薪資 / 法規計算預設仍以台灣規則為主。這不是法律或稅務建議，而是一個希望讓小型團隊能更透明、可審查、可離線使用的開源工具。
