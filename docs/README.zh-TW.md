# 中文專案說明

## 專案定位

Attendance Payroll Offline 是一個離線優先的出勤薪資工具。它可以直接用瀏覽器開啟，不需要伺服器、資料庫或帳號。

目前內建並維護的薪資規則預設以台灣情境為主，包含出勤、假勤、月薪/時薪、勞保、健保、勞退、加班、請假扣薪、薪資單與匯出等流程。

## 為什麼不是直接寫成全球薪資系統？

薪資和工時規則高度依賴地區法規。介面文字可以翻譯，但計算規則不能只靠翻譯就套用到其他國家。

因此本專案採取以下原則：

- UI 與文件可以多語化。
- 台灣薪資規則預設必須保持清楚標示。
- 其他地區使用者可以參考架構，但需要自行確認當地法規。
- 新增其他國家或地區的規則預設時，必須附上文件、來源與測試。

## 適合誰使用？

- 台灣小型公司或工作室，需要一套可以離線保存資料的出勤薪資工具。
- 想研究單一 HTML 離線工具架構的開發者。
- 想協助翻譯、測試、文件整理或改善 UI 的開源貢獻者。

## 不適合什麼用途？

- 不應直接宣稱適用所有國家或地區的薪資法規。
- 不應把輸出結果當作法律、稅務、人資或薪資合規建議。
- 不應將真實員工、薪資、出勤資料提交到公開 GitHub repo。

## 多語化策略

我們歡迎英文與其他語言的文件、UI、錯誤訊息、手冊翻譯。但翻譯不等於法規在地化。

例如：

- "勞保" 可翻成 "Taiwan labor insurance"，但不能翻成其他國家的社會保險制度。
- "健保" 可翻成 "Taiwan national health insurance"，但不能暗示等同於其他國家的醫療保險。
- "勞退" 可翻成 "Taiwan labor pension contribution"，但不能直接套用其他國家的退休金制度。

完整術語請見 [glossary.zh-TW-en.md](glossary.zh-TW-en.md)。
