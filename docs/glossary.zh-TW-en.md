# Bilingual Payroll Glossary

This glossary keeps Taiwan-specific payroll terms explicit. Do not translate these terms into another country's legal concept unless a separate jurisdiction preset exists.

| 中文 | Recommended English | Notes |
|---|---|---|
| 出勤 | Attendance | Work attendance records. |
| 薪資 | Payroll / Salary | Use "payroll" for settlement workflow and "salary" for employee pay. |
| 月薪制 | Monthly salary employee | Employee paid by monthly salary. |
| 時薪制 | Hourly employee | Employee paid by hourly wage. |
| 底薪 | Base salary | Base monthly salary before allowances and deductions. |
| 固定加給 | Fixed allowance | Recurring fixed pay item. |
| 伙食津貼 | Meal allowance | Taiwan payroll context; may not map to another country. |
| 薪資加項 | Payroll addition | Manual positive adjustment. |
| 薪資減項 | Payroll deduction | Manual negative adjustment. |
| 其他費用 | Other expense | Manual item used in settlement. |
| 實發薪資 | Net pay | Final payable amount after additions and deductions. |
| 應發合計 | Gross payable total | Total before deductions in the app's settlement context. |
| 扣款合計 | Deduction total | Total deductions in settlement. |
| 勞保 | Taiwan labor insurance | Keep Taiwan-specific. Do not generalize as "social security". |
| 健保 | Taiwan national health insurance | Keep Taiwan-specific. |
| 勞退 | Taiwan labor pension contribution | Keep Taiwan-specific. |
| 事假 | Personal leave | Usually unpaid in the Taiwan preset. |
| 病假 | Sick leave | Deduction behavior follows the Taiwan preset. |
| 特休 | Annual leave / Special leave | Taiwan labor-law concept; explain if needed. |
| 喪假 | Bereavement leave | Paid handling depends on preset. |
| 補休 | Compensatory leave | Leave accrued from eligible overtime/holiday handling. |
| 國定假日 | National holiday | Taiwan preset holiday handling. |
| 加班 | Overtime | Must be tied to rule preset. |
| 平日加班 | Weekday overtime | Taiwan preset multiplier/rule. |
| 休息日加班 | Rest-day overtime | Taiwan-specific interpretation may vary by settings. |
| 薪資單 | Payslip | Printable payroll statement. |
| 備份 | Backup | User-exported JSON backup. |
| 還原 | Restore | Restore from JSON backup. |

## Translation Rules

- Prefer precise jurisdiction labels over broad approximations.
- If a term has no direct match outside Taiwan, keep "Taiwan" in the English phrase.
- If a contributor proposes a new jurisdiction term, add it under a separate preset rather than changing the Taiwan glossary.
