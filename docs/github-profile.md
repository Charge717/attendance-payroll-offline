## Longer About Text

English:

```text
Attendance Payroll Offline is a portable browser-based attendance and payroll tool for small teams. It supports employee records, attendance, leave, Taiwan-focused payroll settlement, payslip printing, JSON backup/restore, and regression tests for calculation edge cases. Core UI supports multiple languages, but payroll/legal calculation presets are Taiwan-focused by default.
```

中文：

```text
Attendance Payroll Offline 是一套可攜式瀏覽器出勤與薪資管理工具，適合小型團隊使用。功能包含員工資料、出勤紀錄、假勤管理、以台灣規則為主的薪資結算、薪資單列印、JSON 備份還原，以及薪資計算邊界案例測試。核心介面支援多語系，但薪資 / 法規計算預設以台灣規則為主。
```
