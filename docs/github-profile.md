## Short Description

```text

```

```text
離線優先的出勤薪資工具，內建已測試的台灣薪資規則預設，並保留可調整規則的擴充方向。
```

## Longer About Text

```text
Attendance Payroll Offline is a portable browser-based attendance and payroll tool for small teams. It supports employee records, attendance, leave, Taiwan-oriented payroll settlement, payslip printing, JSON backup/restore, and regression tests. The interface and documentation can be multilingual, but payroll rules are jurisdiction-specific and the maintained preset is Taiwan-focused.
```
